1. Создать Angular2+/React приложение “Wiki персонажей вселенной мультсериала Rick & Morty” с использованием публичного API https://rickandmortyapi.com/documentation/#get-all-characters.

Приложение должно состоять из следующих страниц:
- "Список всех персонажей";
- "Детали персонажа";
- "Блок с версткой макета".

Страница "Список всех персонажей" должен отображать список всех персонажей в табличном представлении. Также должна быть возможность сортировать персонажей по имени. По нажатию на персонажа должен открываться экран "Детали персонажа". 

2. Страница "Блок с версткой макета" должен содержать верстку блока из psd-макета.
Требования по верстке:
- Кроссбраузерность - последние версии всех распространенных браузеров;
- Использование CSS3 приветствуется;
- Верстка должна быть гибкой, блок тянется по высоте;
- Perfect Pixel макета, внимание к деталям;
- Написать метод, который при нажатии на кнопку "Продолжить" реализует плавную анимацию прогрессбара от 0 до X%, где Х - случайное целое число от 10 до 100.

При работе с гитом приветствуется создание коммитов на каждую отдельную задачу.
